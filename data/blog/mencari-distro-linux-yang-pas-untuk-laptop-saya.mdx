Alhamdulillah, saya udah upgrade device. Bener-bener kalo udah upgrade device, kerasa banget meningkat produktivitasnya(walau tergantung niat orangnya juga sih, ada juga yang niatnya pengen produktif, tapi karena dia masih main game, malah sibuk dengan gamenya). BTW, laptop saya sekarang Lenovo Thinkbook 14 G2 ARE. 

Dan mungkin saya udah sedikit menyinggung di tulisan sebelumnya, bahwa saya ingin pindah dari Artix ke "Just Works" distro. Dan yaps, saya sudah menemukannya, yaitu KDE Neon.

> Loh, kok Ubuntu based? Kenapa ga Arch based lagi?

Ambil aman eui. Udah nyoba [Arco Linux](https://arcolinux.com/), tapi banyak masalah muncul. Ya namanya hardware baru yak. Juga saya kurang senang banyak Conky. Malah terkesan **"bloat"**.

Nyoba [Artix](https://artixlinux.org/) XFCE via Live USB, tapi sama aja, malah lebih parah ini. Touchpad ga bisa diklik alias cuma bisa digeser doang, muncul suara **beep** keras banget. Tapi selain itu normal sih.

Saya pun teringat dengan suatu postingan di grup Facebook Linux Indonesia, Laptopnya Thinkbook juga. Beliau install Ubuntu tanpa ada masalah. Cuma katanya fingerprintnya aja yang ga bekerja(whatever, ga butuh banget Fingerprint sih, lebih prefer pake password biasa).

Setelah lihat-lihat review distro di [distrowatch](https://distrowatch.com), saya pun mencoba KDE Neon via Live USB. Sesuai ekspektasi, semua bekerja semestinya. Walau memang kernel bawaannya masih 5.4(KDE Neon basednya Ubuntu LTS), yang artinya kita mesti upgrade ke 5.13. 

**Catatan Penting:** semisal udah install KDE Neon nya, cepet masuk ke **<u>Safe Graphics Mode</u>**, karena seperti yang sudah saya bilang tadi, dia masih pake kernel 5.4, kita mesti mengupgrade nya dulu ke 5.13. Kalo kita malah booting normal, bakal muncul tampilan seperti tv rusak(You know what i mean).
